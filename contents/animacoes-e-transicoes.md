# 🧑‍🔬 Animações e Transições
Adicione movimentos suaves aos elementos.

Exemplo de transição:
```css
button {
    transition: background-color 0.5s ease;
}

button:hover {
    background-color: coral;
}
```

Exemplo de animação:
```css
@keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
}

.box {
    animation: fadeIn 1s ease-in-out;
}
```
