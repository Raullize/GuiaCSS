# 🖼️ Background e Border
- **Background** define a cor ou imagem de fundo de um elemento.
- **Border** adiciona bordas ao redor do elemento.

Exemplo:
```css
div {
    background-color: lightblue;
    border: 2px solid black;
    border-radius: 5px;
}
```
