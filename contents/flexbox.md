# 📐 Flexbox
Modelo de layout flexível ideal para alinhamento e distribuição de espaço.

Exemplo básico:
```css
.container {
    display: flex;
    justify-content: center; /* Alinha no eixo principal */
    align-items: center; /* Alinha no eixo cruzado */
}
```
