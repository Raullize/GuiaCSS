# 📏 Unidades de Medida
- **Absolutas**: `px`, `cm`, `mm`, `in` (não se ajustam ao tamanho da tela).
- **Relativas**: `em`, `rem`, `%`, `vw`, `vh`, `vmin`, `vmax` (se ajustam dinamicamente).

Exemplo:
```css
.container {
    width: 80vw; /* 80% da largura da tela */
    font-size: 1.5rem; /* 1.5 vezes o tamanho da fonte raiz */
}
```
